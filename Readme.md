# 🧠 AI Mental Fatigue Detection System

An AI-powered web application that detects mental fatigue using **face and voice analysis**.  
It uses computer vision and machine learning techniques to provide real-time fatigue status and score.

---

## 🚀 Features

- 🎥 Real-time face fatigue detection (Webcam)
- 🎙️ Voice analysis using MFCC features
- 🤖 Machine Learning-based prediction
- 📊 Fatigue status: **Normal / Fatigued**
- 📈 Fatigue score (%) display
- 🌐 Interactive web interface

---

## 🛠️ Tech Stack

- **Backend:** Python, Flask  
- **Computer Vision:** OpenCV, MediaPipe  
- **Machine Learning:** Scikit-learn  
- **Audio Processing:** Librosa  
- **Frontend:** HTML, CSS, JavaScript
